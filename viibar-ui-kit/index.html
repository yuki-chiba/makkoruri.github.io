Viibar UI Kit Components & Style Guide

Buttons

Main Action Buttons（通称Main）

メインアクションに使うボタン。出来る限り1画面に2個以上おかないようにしてください。


<button type="button" class="btn btn-main">確定する</button>

Main Tool Buttons（通称poco）

メインアクションではありませんが、主要な機能に使うボタンスタイルです。何個でも置いてOKです。アイコンも一緒に使うことが多いです。


<button type="button" class="btn btn-poco"><i class="fa fa-download"></i> ダウンロード</button>

sub Buttons（通称peta）

補助的な機能やモーダルの閉じるボタン、キャンセルボタンに使うボタンスタイルです。


<button type="button" class="btn btn-peta">キャンセル</button>

delete Buttons

削除するボタンに使うボタンスタイルです。ユーザーに危険を知らせるためなら削除以外でも使ってOKです。


<button type="button" class="btn btn-peta btn-emergency">削除する</button>

Button Size

Normal Buttons

一番使う頻度の高い通常サイズです。デフォルトサイズなのでサイズ指定の記述は特にいりません。


<button type="button" class="btn btn-main">Normal</button>

Mini Buttons

一番小さいサイズのボタンです。


<button type="button" class="btn btn-main btn-mini">Mini</button>

Small Buttons

あまり目で立たなくてもいい補助的なボタンのサイズです。


<button type="button" class="btn btn-main btn-sm">Small</button>

Large Buttons

目立たせたい場合に使う大きめサイズのボタンです。


<button type="button" class="btn btn-main btn-lg">Large</button>

King Buttons

見逃してもらっては困る！確実に目ただせたい場合に使うキングサイズのボタンです。


<button type="button" class="btn btn-main btn-king">King</button>
